# MODAnime 
### v1.0.1

Start electron along with the internal server (in view mode)
```bash
npm start
```

Run project in dev mode with live loading
```bash
npm run dev
```

Build Svelte and Tailwind
```bash
npm run build
```

Build Svelte and Tailwind for production
```bash
npm run build:prod
```

Run ESlint
```bash
npm run fix
```

Run Prettier
```bash
npm run format
```
